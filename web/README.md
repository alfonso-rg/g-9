# Laboratorio de monedas trucadas

Aplicacion web docente para entender de forma visual la relacion entre:

- significatividad
- potencia
- error tipo I
- error tipo II
- tamano muestral

La app usa un contraste binomial unilateral sobre monedas sesgadas y combina simulacion de muestras con visualizaciones de distribuciones y curva de potencia.

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Despliegue recomendado

Netlify, usando la configuracion del fichero `../netlify.toml`.
