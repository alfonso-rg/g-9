# Curso y laboratorio interactivo

Este repositorio contiene los materiales del curso de diseno de experimentos y una web docente para explorar significatividad, potencia, errores tipo I y II, y tamano muestral con monedas trucadas.

## Estructura

- `main_25_26.tex` y ficheros relacionados: material del curso.
- `web/`: aplicacion React + TypeScript + Vite.
- `netlify.toml`: configuracion de despliegue recomendada en Netlify.

## Desarrollo local de la web

```bash
cd web
npm install
npm run dev
```

## Build de produccion

```bash
cd web
npm run build
```
